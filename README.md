# Thread Vibe — Fashion Storefront Demo

A modern fashion e‑commerce demo built with **Next.js** and **Tailwind CSS**.  
Thread Vibe showcases curated collections, premium visuals, and a seamless shopping flow.  
Designed for performance, accessibility, and a bold, contemporary retail experience.

---

## Features

- Curated product collections with category filtering
- Smooth shopping flow with cart and checkout integration
- Responsive layouts optimized for mobile and desktop
- Premium product visuals with modern animations
- Accessibility‑first design with SEO‑friendly structure

---

## Tech Stack

- [Next.js](https://nextjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [TypeScript](https://www.typescriptlang.org/)
- [Framer Motion](https://www.framer.com/motion/)
- [Supabase](https://supabase.com/)
